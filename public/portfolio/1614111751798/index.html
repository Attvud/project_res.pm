Textpool.ruWebsite for a copywriting exchange

Detailed information

Watch in: InVision

I implemented this project with a rather complex interface in two weeks.

Scope of the project:

  • 80 screens for the desktop version
  • 20 screens for mobile version

Distinctive features

  • service visitors can be both customers and clients. The user can switch between these roles at any time
  • a two-sided personal account is implemented in such a way that Front-end developers have as little work as possible
  • system of moderation of works performed by authors
  • automatic quality control system for authors' works
  • management of the main functionality in one or two clicks
  • statistics of earned / spent funds are always in front of your eyes
  • minimum work to implement the mobile version of the project
  • filters and settings are moved to convenient sidebars that appear when needed and do not load the main interface
